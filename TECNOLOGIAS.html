TECNOLOGÍAS

Las tecnologías en el Front-End son el conjunto de lenguajes, herramientas y frameworks que se utilizan para crear la parte visual e interactiva de una página web o aplicación: es decir, todo lo que el usuario ve y con lo que puede interactuar.

  • HTML
  • CSS
  • JAVA SCRIPT

Estas tecnologías no solo se enfocan en mostrar información, sino en garantizar que la experiencia de navegación sea agradable, rápida, intuitiva, adaptable a diferentes dispositivos (responsive design) y accesible para todo tipo de usuarios.

El Front-End funciona como un puente entre el Back-End (la parte lógica y de servidor) y el usuario final, transformando datos y procesos en elementos visuales fáciles de comprender e interactuar.

HTML

HTML (HyperText Markup Language) es el lenguaje estándar de marcado utilizado para crear y estructurar páginas web.

No es un lenguaje de programación, sino un lenguaje de etiquetas que define la estructura del contenido: títulos, párrafos, imágenes, enlaces, botones, formularios, tablas, listas, etc.

Es la base fundamental de cualquier sitio web.

CSS

CSS (Cascading Style Sheets), en español Hojas de Estilo en Cascada, es el lenguaje utilizado para dar estilo y presentación a las páginas web. Mientras que HTML define la estructura (títulos, párrafos, botones, imágenes), el CSS se encarga de definir la apariencia visual: colores, tamaños, fuentes, márgenes, posiciones, animaciones y diseño adaptable a diferentes dispositivos (responsive design).

JS

JavaScript (JS) es un lenguaje de programación interpretado, de alto nivel, dinámico y multiparadigma, creado en 1995 por Brendan Eich en Netscape, con el propósito de dotar a las páginas web de interactividad y dinamismo. Es considerado uno de los tres pilares fundamentales del desarrollo web, junto con HTML (estructura) y CSS (diseño). Mientras que HTML se encarga de definir qué se muestra en una página y CSS cómo se ve, JavaScript controla el “cómo se comporta”. Gracias a él, los sitios web pueden reaccionar a las acciones del usuario, manipular datos en tiempo real, validar formularios, mostrar animaciones, actualizar contenido sin recargar la página y mucho más.
